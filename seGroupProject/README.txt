Directions on how to run the racecar game

Step 1
Load up xampp control and start the mysql server

Step 2
Navigate to the xampp/mysql/bin directory

Step 3
Enter the command:
	mysql -h localhost -u student -p
Then enter the password:
	hello

Step 4
Enter the command:
	use student_space;
The enter the command:
	source (Location of project.sql file)
Example
	source C:/temp/project.sql;

Step 5
Load up papyrus

Step 6
Import all of the files of the project into papyrus

Step 7
Run the bat server bat file

Step 8
Within the server window click the listen button

Step 9
Run the bat client file
